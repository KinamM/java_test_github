Для ініціалізації репозиторія - пишемо git init .

Щоб зрозуміти поточний стан репозиторія - пишемо git status

Щоб добавити файл в staging area - пишемо git add <імя файлу>

Щоб зробити комміт - пишемо git commit -m "<опис комміту>"

//Щоб подивитися історію комітів - пишемо git log

Щоб перейти до конкретного коміту - пишемо git checkout <hash>

Щоб залити віддалений репозиторій на github - пишемо git remote add origin <адрес репозиторія - куди залити, беремо з github>

Щоб подивитися список віддалених репозиторіїв - пишем git remote --v

щоб залити код на віддалений репозиторій - пишемо git push origin master

Щоб підтягнути код з віддаленого репозиторія - пишемо git pull origin master


ЯК ПРАЦЮВАТИ З ГІЛКАМИ:

git branch - показує на якій я гілці зараз
git checkout -b <branch_name> - створює нову гілку з іменем <branch_name> і переключається на неї
git checkout <branch_name> - переключається на гілку з вказаним імям
git branch -d <branch_name> - видалити вказану гілку
git merge <branch_name> - злити код з вказаної гілки в ту де знаходимося
